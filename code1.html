Installation of Ruby and Rails

Before installing rails you should check if you have Ruby. To see what version of ruby you are currently running you would run the following command in whatever bash or terminal you are using. Ruby on rails developement relies heavily on a terminal tools.

$ ruby -v

Pressing enter will output the version of Ruby that is currently installed on your machine. If you recieve an error or have a version of ruby ealier than 2.2 you will need to get a more up to date version of ruby

Now that you have installed the most recent version of the Ruby making sure that you are up to date and ready to install rails!

To get Rails and you have Ruby you can run the command:

$ gem install rails

This will install the latest version of rails.

Now you are ready to create rail apps!
