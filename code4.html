Making a Controller

To create a controller you go into the terminal and run the script

$rails generate controller controllerName

This command will create 2 things. One, a controller in the file apps/controller/controller_name.rb. And two, a view file in apps/view/controller_name.

In the apps/controller/controller_name.rb file you will put code for the creation making the controller modify the given view. When it is first created it looks like this.

class PagesController < ApplicationController

end

This is a cimple class declaration of a controller it does nothing at the moment, but if we add the code:

class PagesController < ApplicationController
def home
end
end

This will direct the program to go to the view home. But before the program can go there we need to create the view.
