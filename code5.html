Making a View

With the router and the controller set up we can now make a view!

In the apps/views/controller_name file that was created when we made a controller we can create a file that will work as our view. This file needs to be saves with the extension html.erb. This extension allows the program to make the HTML with the possible Ruby elements that might be inline with the HTML.

Also with this file type you don't need to specifiy the DOCTYPE or The HEAD or the BODY. These usually needed HTML elements are already taken care for us as part of the Rails framework and we will go over that in a later section.

<div id="main">
<p> This is a Rail App! </p>
</div>

This will do for now, and the Rail App should work now! It doesn't do much but it works. We will get it to do some stuff later.
