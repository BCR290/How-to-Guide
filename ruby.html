What's Ruby?

Definition

A powerful, flexible programming language in web/Internet development.

Background

Ruby was designed by Yukihiro Matsumoto (often just called "Matz") in 1995. It is designed to emphasize human needs over the computer, which is why Ruby is so easy to pick up.

Functionalities

process text

create games

part of the popular Ruby on Rails web framework

Properties

High-level, -reading and writing Ruby is really easy, it looks a lot like regular English!

Interpreted, -don't need a compiler to write and run Ruby. You can write it on your own computer!

Object-oriented -allows users to manipulate data structures called objects in order to build and execute programs.
