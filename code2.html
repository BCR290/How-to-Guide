Model View Controller and Router Locations

So lets get aquainted with the files that you will be looking at. When you create a rails app you always start with some basic commands:

1.
$ gem install rails
of course you can name your App whatever you like but I am going to use the MySite for this Demo
2.
$ cd MySite
$ bundle install
cd - puts you into your apps file
and the other command installs the necesary files

Now that you have all the proper files to start developing an app lets take a look at them. At first it may seem a little intimidating but with practice and hard work you can learn and become comfortable with navagating this jungle of files.

The three files that you really need to look at are the

  1. app
    • app/controller = your controllers
    • app/view = your veiws
  2. config
    • config/routes.rb = your routes
  3. db = model (databases)

That is the basics of the files that we will be working with.
